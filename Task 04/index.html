CS Technical Glossary

HTML (HyperText Markup Language):
The standard language used to create the structure of web pages.
Algorithm:
A step-by-step set of instructions used to solve a specific problem or perform a task.
Variable:
A reserved memory location used to store data that can be changed during program execution.
Compiler:
A special program that translates high-level source code (like C++) into machine code that a computer understands.
Array:
A collection of items stored at contiguous memory locations, usually of the same data type.
Boolean:
A data type that has one of two possible values: true or false.
Syntax:
The set of rules that defines the combinations of symbols that are considered to be correctly structured programs in a language.
Loop:
A sequence of instructions that is continually repeated until a certain condition is reached.
Function:
A block of organized, reusable code that is used to perform a single, related action.
Pointer:
A variable whose value is the address of another variable, commonly used in C++.
Linked List:
A linear data structure where elements are not stored at contiguous memory locations; instead, they are linked using pointers.
DBMS (Database Management System):
Software for creating and managing databases, such as the one I'm building for UOG Medical Center.
Primary Key:
A specific choice of a minimal set of attributes that uniquely specify a tuple (row) in a relation (table).
CSS (Cascading Style Sheets):
The language used for describing the presentation of a document written in HTML.
Frontend:
The part of a website or software that the user interacts with directly (the "Client-side").
CPU (Central Processing Unit):
The primary component of a computer that acts as its "brain," performing calculations and executing instructions.
IP Address:
A unique string of numbers separated by periods that identifies each computer using the Internet Protocol to communicate over a network.
